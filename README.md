## ✨ الصفحات والميزات

### 🏠 شاشة البداية (Splash)
- تأثير جسيمات متحركة
- شريط تحميل بأربع مراحل (أمان، قاعدة بيانات، خرائط، QR)
- حلقات نابضة وأيقونة درع متوهجة

### 📊 لوحة التحكم
- 4 بطاقات إحصائية (الإجمالي، اليوم، العاجلة، المكتملة)
- مخطط دائري لتوزيع التقارير
- مخطط أعمدة للأولويات
- جدول آخر 10 تقارير مع بحث سريع
- 8 إجراءات سريعة

### 📝 إنشاء التقارير
- **12 نوع تقرير**: عمليات، استخبارات، لوجستي، طبي، استطلاع، اتصالات، هندسي، أمني، نهاية مهمة، مذكرة، يومي، حادثة
- **5 درجات سرية**: غير سري → سري جداً → سري للغاية
- **4 درجات أولوية**: روتيني → أولوية → فوري → فلاش (يومض)
- ترقيم تلقائي: `OPS-202603-1001`
- معاينة مباشرة أثناء الكتابة
- خيارات: QR، باركود، علامة مائية، شريط أمان، ختم
- 3 توقيعات: محرر، مراجع، معتمد
- تكبير/تصغير المعاينة

### 🗂️ أرشيف المستندات
- عرض شبكي أو قائمة
- بحث بالعنوان/الرقم/الوحدة
- تصفية بالنوع والأولوية
- إجراءات: عرض، طباعة، نسخ، حذف
- قائمة سياقية (كليك يمين)

### 📋 مكتبة القوالب
- 12 قالب جاهز مع معلومات الأولوية والسرية
- تحميل القالب وملء البيانات مباشرة

### 🗺️ الخرائط العسكرية
- رسم تفاعلي على قماش Canvas
- **10 رموز عسكرية**: قوات صديقة/معادية، هدف، قاعدة، نقطة تفتيش، إمدادات، طبي، مدفعية، مطار، مركز قيادة
- **7 أدوات رسم**: تحديد، تحريك، ممحاة، خط، سهم، خط منقط، مستطيل، دائرة، رسم حر، نص
- تحكم في اللون والسمك والشفافية
- طبقات يمكن إظهار/إخفاؤها
- بوصلة متحركة ومؤشر إحداثيات
- تصدير PNG

### 🔵 مصمم الأختام
- **5 أشكال**: دائري، بيضاوي، مستطيل، سداسي، معيني
- **4 أنماط**: كلاسيكي، حديث، مزخرف، بسيط
- نص دائري علوي وسفلي، متن رئيسي، وحدة/تاريخ
- 0، 3، أو 5 نجوم
- **8 أختام جاهزة**: سري، سري جداً، رسمي، معتمد، عاجل، مصنف، طبي، خاص
- تحميل PNG

### 📄 مصمم المستندات
- **5 قوالب جاهزة**: عمليات، استخبارات، خطاب رسمي، مذكرة داخلية، شهادة تقدير
- **12 عنصر قابل للإضافة**: ترويسة، عنوان، قسم نصي، جدول، توقيعات، QR، ختم، فاصل، صورة، علامة مائية، شريط أمان، تذييل
- نص قابل للتحرير مباشرة
- تحكم في التكبير (40% — 160%)
- طباعة وتصدير PDF

### 📱 مدير QR والباركود
- **رمز QR**: تخصيص الحجم (128-350)، مستوى التصحيح (L/M/Q/H)، اللون
- **باركود**: Code 128 مع نص مخصص
- **علامة مضادة للتزوير**: 3 أنماط (خطوط، نقاط، موجات)
- **بطاقة هوية عسكرية**: توليد بطاقة بالاسم والرتبة والرقم

### 📈 الإحصاءات
- مخطط أعمدة: التقارير حسب النوع
- مخطط دائري: التوزيع النسبي
- مخطط أولويات: روتيني → فلاش
- مخطط سرية: توزيع درجات السرية

### ⚙️ الإعدادات
- بيانات المؤسسة (الاسم، القيادة، البلد)
- تخصيص المظهر (اللون، الخط، الثيم)
- كلمة مرور النظام
- إعدادات الطباعة (الحجم، الاتجاه)
- تصدير/استيراد JSON
- مسح البيانات
